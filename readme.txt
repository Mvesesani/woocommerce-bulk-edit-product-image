=== MVSN WooCommerce Bulk Set Image ===
Tags: woocommerce, product, bulk, image upload, product image
Requires at least: 5.4
Tested up to: 5.4

Easily bulk set product images in the WooCommerce product bulk edit screen.

== Description ==

Great for product image management in WooCommerce!
This plugin adds an image selection option to the WooCommerce product bulk edit screen.
Saves a tremendous of time!

== Installation ==


1. Upload th zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Products", select some products and choose to bulk edit them where you can select an image to be set for multiple products.
